Web aplikacija (PostgreSQL/.NET/React)

Autor: Teo Matijašić

Web aplikacija   |   2022. - 2023.   |   .NET   |   React |   PostgreSQL

Opis Projekta

Ovaj projekt je podrazumijevao izradu web aplikacije za evidenciju stručne prakse na fakultetu. Pritom su kreirani svi osnovni dijelovi web aplikacije, od baze podataka u PostgreSQL-u, srednjeg dijela u C# i .NET-u te frontenda u React-u.

Cilj i Problem Projekta

Glavni cilj ovoga projekta bio je kreirati web aplikaciju za evidenciju stručne prakse na fakultetu kako bi se digitaliziralo popisivanje studenata, tvrtki i njihovih stručnih praksi.

Rezultati i Moja Uloga

Kreirana je vrlo detaljna i korisna web aplikacija koja sadrži brojne mogućnosti, od prijave koja različitim korisnicima dodaje zasebne privilegije, ovisno o ulozi, pa sve do mogućnosti popisivanja svih tvrkti, studenata i njihovih stručnih praksi kako bi sve te informacije bile na jednom mjestu. Web aplikaciju sam razvio kao glavni i jedini član tima.
